# Mechanics Models

Constitutive models for finite element simulations.

## Modules

### [cohesive](./cohesive.md)
Cohesive zone interface for matrix-free fracture simulations. Provides `CohesiveInterface` for cohesive element assembly, along with `exponential_potential` and `bilinear_potential` traction-separation laws.
